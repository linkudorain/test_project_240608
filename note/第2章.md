# 2-1

## 塑性ひずみ

### 1. 塑性ひずみの定義:
   塑性ひずみ $\varepsilon^p$ は全ひずみ $\varepsilon$ から弾性ひずみ $\varepsilon^e$ を引いたものとして定義される:

   $\varepsilon = \varepsilon^e + \varepsilon^p$

### 2. 弾性ひずみと応力の関係:
   弾性ひずみは応力 $\sigma$ とヤング率 $E$ を用いて表される:

   $\varepsilon^e = \frac{\sigma}{E}$

### 3. 塑性ひずみの計算例:
   - ヤング率 $E = 200,000 \text{ MPa} = 200 \text{ GPa}$
   - 塑性ひずみの計算式: $\varepsilon^p = \varepsilon - \frac{\sigma}{E}$
   - 例1: $\varepsilon = 0.01 \text{ (1.0%)}$, $\sigma = 300 \text{ MPa}$ の場合
     $\varepsilon^p = 0.01 - \frac{300}{200000} = 0.0085 \text{ (0.85%)}$
   - 例2: $\varepsilon = 0.005 \text{ (0.5%)}$, $\sigma = 244 \text{ MPa}$ の場合
     $\varepsilon^p = 0.005 - \frac{244}{200000} = 0.0038 \text{ (0.38%)}$

### 4. 注意点:
   - 弾塑性変形が進行すると、弾性係数は変化する可能性がある
   - 塑性変形後の弾性係数 $E$ は初期値とは異なる場合がある

# 2-2

## 降伏条件

### 1. 降伏条件の種類:
   塑性変形の開始を表す初期降伏条件には以下のようなものがある：
   - ミーゼス（von Mises）の条件
   - トレスカ（Tresca）の条件
   - モール・クーロン（Mohr-Coulomb）の条件（脆性材料）
   - ドラッカー・プラガー（Drucker-Prager）の条件

### 2. 条件の特徴:
   - トレスカの条件：最大せん断応力に基づく
   - ミーゼスの条件：せん断ひずみエネルギーまたは八面体せん断応力に基づく
   - モール・クーロンの条件：土木分野で多用される

### 3. 適用例:
   - 静水圧の影響を受けない場合の初期降伏条件として使用される
   - ドラッカー、プラガーの条件は土質材料などの摩擦性材料の降伏挙動とともに塑性体積変化の表示に用いられる

# 2-3

## 降伏条件

### 1. ミーゼスの降伏条件:
   3次元主応力空間で表される式：
   
   $\frac{1}{2}[(\sigma_1 - \sigma_2)^2 + (\sigma_2 - \sigma_3)^2 + (\sigma_3 - \sigma_1)^2] = Y^2$

   ここで、$Y$ は単軸引張試験での降伏応力。

### 2. 特徴:
   - 静水圧成分に依存しない（$\sigma_1 = \sigma_2 = \sigma_3 = 0$ とすると $\sigma = Y$ で降伏）
   - せん断応力 $\tau$ が作用する場合、主応力は $\frac{Y}{\sqrt{3}}$ で降伏する

### 3. 主応力表示:
   $\sigma_1 - \sigma_2 = \frac{1}{\sqrt{2}}Y$
   $\sigma_2 - \sigma_3 = \frac{1}{\sqrt{2}}Y$
   $\sigma_3 - \sigma_1 = -\sqrt{2}Y$

### 4. トレスカの降伏条件との比較:
   - ミーゼスの条件は主応力平面では円となる
   - トレスカの条件は主応力軸上に頂点を持つ六角形となる
   - ミーゼスの円に内接する正六角形がトレスカの条件

### 5. 注意点:
   - 実験により求められる降伏応力は、これらの条件の中間にあることが多い
   - 材料や負荷条件によって適切な降伏条件を選択する必要がある

# 2-4

## 降伏条件

### 1. 平面応力状態の降伏条件:
   静水応力（八面体垂直応力）に無関係な応力を示すことになる。

### 2. トレスカの降伏条件:
   - 主応力の最大差が材料固有の値になったときに降伏
   - 最大せん断応力説とも呼ばれる

### 3. ミーゼスの降伏条件:
   - 八面体せん断応力が材料固有の値になったときに降伏
   - トレスカの降伏条件の外接円として表される

### 4. π平面上での表示:
   - π平面に垂直な方向は $(1,1,1)$ の方向
   - ξ軸は $(1,\sqrt{3},1/\sqrt{3})$ の方向
   - η軸は $(1,-1/\sqrt{3},1/\sqrt{3})$ の方向
   - ζ軸は $(0,2/\sqrt{3},-1/\sqrt{3})$ の方向

### 5. 主応力の表示:
   $\sigma_1 = \frac{1}{\sqrt{2}}\xi + \frac{1}{\sqrt{6}}\eta + \frac{1}{\sqrt{3}}\zeta$
   $\sigma_2 = -\frac{1}{\sqrt{6}}\eta + \frac{1}{\sqrt{3}}\zeta$
   $\sigma_3 = -\frac{1}{\sqrt{2}}\xi + \frac{1}{\sqrt{6}}\eta + \frac{1}{\sqrt{3}}\zeta$

### 6. ミーゼスの降伏条件（π平面上）:
   $\eta^2 + \xi^2 = (\frac{2}{\sqrt{3}}Y)^2$

### 7. トレスカの降伏条件（π平面上）:
   $\eta = \pm \frac{1}{\sqrt{3}}Y$
   $\eta = \pm \frac{1}{\sqrt{3}}(\xi + \sqrt{3}Y)$
   $\eta = \pm \frac{1}{\sqrt{3}}(\xi - \sqrt{3}Y)$

# 2-5

## モール・クーロン降伏条件

### 1. 概要:
   土や岩などの材料強度は静水圧に依存する。モール・クーロン降伏条件は、この依存性を考慮した降伏条件として用いられる。

### 2. 数学的表現:
   $\tau = f(\sigma)$
   ここで、$\tau$ はせん断応力、$\sigma$ は垂直応力。

### 3. 一般的な表現:
   $\tau = (tan \phi)\sigma + c$
   ここで、$\phi$ は内部摩擦角、$c$ は粘着力。

### 4. 特徴:
   - 二次元では直線で表される
   - 三次元では円錐面となる

### 5. 主応力空間での表現:
   $\sigma_1 = K_p\sigma_3 + 2c\sqrt{K_p}$
   ここで、$K_p = tan^2(45^\circ + \frac{\phi}{2})$

### 6. π平面上での表現:
   六角形の降伏曲面となり、静水圧軸に沿って拡大する。

### 7. 適用範囲:
   - 土質力学や岩盤力学で広く使用される
   - 摩擦性材料の降伏挙動を表現するのに適している

### 8. 注意点:
   - 静水圧依存性を考慮しているため、金属材料よりも土や岩などの材料に適している
   - 材料によっては、より複雑な降伏条件（例：Drucker-Prager条件）が必要な場合がある

### 9. 図解:
   画像には、モール・クーロン降伏条件を表す曲線が示されている。縦軸に$\tau$（せん断応力）、横軸に$\sigma$（垂直応力）をとり、直線で表される降伏条件が描かれている。

# 2-6

## 降伏条件（静水圧の影響）

### 1. 降伏条件の種類:
   - ミーゼス（von Mises）の条件
   - トレスカ（Tresca）の条件
   - モール・クーロン（Mohr-Coulomb）の条件
   - ドラッカー・プラガー（Drucker-Prager）の条件

### 2. 条件の特徴:
   - ミーゼスとトレスカの条件: 静水圧の影響を受けない
   - モール・クーロンの条件: $\tau = f(\sigma)$ の包絡線として表現され、クーロンの式 $(\tau = c + \sigma \tan \theta)$ を採用
   - ドラッカー・プラガーの条件: ミーゼスの降伏条件を拡張し、材料の静水圧依存性を考慮

### 3. 適用範囲:
   - ミーゼス、トレスカ: 主に金属材料に適用
   - モール・クーロン、ドラッカー・プラガー: 土質材料など、静水圧の影響を受ける材料に適用

# 2-7

## 等方硬化則

### 1. ヤング率の計算例:
   $E = \frac{200}{0.001} = 200000 \text{ MPa} = 200 \text{ GPa}$

### 2. 等方硬化の特徴:
   - 塑性変形後もヤング率は変化しないと仮定
   - 加工硬化により降伏曲面の原点は移動せず、半径が増大
   - 例: 初期降伏応力が 300MPa で、塑性ひずみが 1.0% の場合の新しい降伏応力を計算

### 3. 計算例:
   $\varepsilon^p = 0.01 - \frac{300}{200000} = 0.0085 \text{ (0.85%)}$
   新しい降伏応力 = 300 + (-300) = 600 MPa
   全ひずみは 1.0% - 0.3% = 0.7% となる

### 4. グラフ解釈:
   - 横軸: ひずみ (%)
   - 縦軸: 応力 (MPa)
   - 初期降伏点: (0.1%, 200 MPa)
   - 新しい降伏点: (1.0%, 300 MPa)
   - 除荷後の永久ひずみ: 0.7%

# 2-8

## 移動硬化則

### 1. 概要:
   - 降伏曲面の中心が移動する硬化則
   - バウシンガー（Bauschinger）効果を表現可能

### 2. 特徴:
   - 降伏曲面の形状は変化しない
   - 中心が変形とともに移動する（背応力）

### 3. モデリング手法:
   - 降伏条件の表示に対する中心点を導入
   - 背応力とともに移動していくようにモデリング

### 4. バウシンガー効果:
   - 塑性変形後、逆方向の降伏応力が小さくなる現象
   - 移動硬化則を用いることで表現可能

# 2-9

## 移動硬化則

### 1. 定義:
   降伏曲面の中心点である背応力が変化すると考える移動硬化則と降伏曲面の大きさも変化する等方硬化則がある。

### 2. 単軸線形移動硬化則の特徴:
   - 上下方向の応力-ひずみ関係は直線的
   - 除荷時も同様の傾きで直線的に除荷

### 3. 数学的表現:
   $|\sigma - \alpha| = Y$
   ここで、$Y$ は降伏応力、$\alpha$ は背応力。

### 4. 挙動:
   - 正方向に引張変形すると $\sigma = Y + \alpha$
   - 逆方向に圧縮変形すると $\sigma = -Y + \alpha$
   - 降伏応力の差は常に $2Y$ となる

### 5. バウシンガー効果:
   - ①では初の反転点前において $\sigma = Y$ で降伏している
   - ②と③では最初の除荷点から $2Y$ 離れた点で降伏する

### 6. 注意点:
   - 実際の材料挙動と完全には一致しない
   - より複雑なモデルが必要な場合もある

# 問2-10 硬化則

## 1. 概要:
   応力変動を伴う塑性変形を記述するためには、降伏曲面が塑性変形の進行に伴ってどのように変化するかを記述する規則（硬化則）が必要となる。

## 2. 硬化則の種類:
   - 等方硬化則
   - 移動硬化則
   - 複合硬化則

## 3. 等方硬化則:
   - 降伏曲面の中心位置が変化せず、大きさのみが変化するモデル
   - バウシンガー効果を表現できない
   - 単調負荷に近い場合は実際とよく合う
   - 有限要素法(FEM)解析で広く使用されている

## 4. 移動硬化則:
   - 降伏曲面の大きさは変化せず、中心位置が移動するモデル
   - バウシンガー効果を表現できる
   - 繰り返し負荷を受ける場合に適している

## 5. 複合硬化則:
   - 等方硬化と移動硬化の効果を組み合わせたモデル
   - より広範囲の塑性変形挙動を表現可能

## 6. 数学的表現:
   等方硬化則の降伏関数は:
   $f = \frac{3}{2}σ_{ij}σ_{ij} - [σ_y(ε_p)]^2 = 0$
   
   移動硬化則の降伏関数は:
   $f = \frac{3}{2}(σ_{ij} - α_{ij})(σ_{ij} - α_{ij}) - σ_y^2 = 0$

   ここで、$ε_p$ はミーゼスの相当塑性ひずみ、$α_{ij}$ は背応力（back stress）と呼ばれる。

## 7. 複合硬化則の表現:
   $f = \frac{3}{2}(σ_{ij} - α_{ij})(σ_{ij} - α_{ij}) - [σ_y(ε_p)]^2 = 0$

## 8. 注意点:
   - 実際の材料挙動は、これらのモデルよりも複雑な場合が多い
   - 適切な硬化則の選択は、対象とする材料や負荷条件によって異なる

## 9. 応用:
   - 金属材料の塑性変形解析
   - 構造物の強度評価
   - 材料の疲労寿命予測

# 問2-11 流れ則

## 1. 概要:
   塑性ひずみを比べる。弾性ひずみはどの経路でも同じであるが、プラントル・ロイス（Prandtl-Reuss）の式から塑性ひずみの速度成分は以下のように与えられる：

   $\dot{\varepsilon} = \frac{\sigma}{E} + \frac{2}{3} \frac{\bar{\sigma}}{GH} \sigma$

   ここで第1項は弾性成分、$\bar{\sigma}$ は相当応力の速度、$\sigma$ は偏差応力、$H$ は加工硬化係数、$G$ は剛性率の2倍成分である。

## 2. 塑性ひずみの計算:
   これを経路に沿って積分する。弾性ひずみは $Y$ を降伏応力として $0 < \sigma < Y$ の範囲で考える。塑性ひずみ速度については経路ごとに異なる。

## 3. 経路A:
   $\tau < Y/\sqrt{3}$ の折れ曲がる前では、塑性ひずみ $\varepsilon = 0$
   折れ曲がった後は、相当応力 $\bar{\sigma} = \sqrt{\sigma^2 + Y^2}$
   相当応力の速度 $\dot{\bar{\sigma}} = \sigma\dot{\sigma} / \sqrt{\sigma^2 + Y^2}$
   塑性ひずみの速度 $\dot{\varepsilon} = \frac{\dot{\sigma}}{E} + \frac{1}{H} \frac{\sigma^2\dot{\sigma}}{\sigma^2 + Y^2}$

   上式を積分すると：
   $\varepsilon_A = \int \frac{d\sigma}{E} + \frac{1}{H} \int \frac{\sigma^2 d\sigma}{\sigma^2 + Y^2} = \frac{Y}{E} + \frac{Y}{H} (1 - \frac{\pi}{4})$

## 4. 経路B:
   比率 $\sigma : \tau = 1 : \frac{1}{\sqrt{3}}$ を使うので、相当応力 $\bar{\sigma} = \sqrt{2}\sigma$、相当応力の速度 $\dot{\bar{\sigma}} = \sqrt{2}\dot{\sigma}$ となる。
   降伏するときは $Y = \sqrt{2}\sigma < Y$ の区間となる。

   塑性ひずみの速度 $\dot{\varepsilon} = \frac{\sigma}{E} + \frac{\sigma}{H}$
   積分範囲に注意して積分すれば：
   $\varepsilon_B = \frac{Y}{E} + \frac{Y}{H} (1 - \frac{1}{\sqrt{2}})$

## 5. 経路C:
   折れ曲がる前までは弾性で塑性ひずみ $\varepsilon = \sigma / E$
   折れ曲がった後の相当応力 $\bar{\sigma} = \sqrt{Y^2 + 3\tau^2}$
   相当応力の速度 $\dot{\bar{\sigma}} = \frac{3\tau\dot{\tau}}{\sqrt{Y^2 + 3\tau^2}}$
   
   折れ曲がった後の塑性ひずみの速度：
   $\dot{\varepsilon} = \frac{Y}{H} \frac{3\tau\dot{\tau}}{Y^2 + 3\tau^2}$
   
   積分範囲に注意して積分すれば：
   $\varepsilon_C = \frac{Y}{E} + \frac{Y}{H} [\frac{Y^2\tau^2}{Y^2 + 3\tau^2} + \frac{1}{E} + \frac{1}{6} \ln \sqrt{2}]$

# 2-12

## 流れ則

### 1. 概要:
   塑性変形時における応力状態と塑性ひずみ速度状態の関連を示す法則。塑性ポテンシャルの概念を用いて関連流れ則と呼ばれる。

### 2. 塑性ポテンシャル:
   - 塑性ひずみ速度を決定するための関数
   - 一般に降伏関数Fと同じものを用いる（関連流れ則）

### 3. 数学的表現:
   $\dot{\varepsilon}_{ij}^p = \lambda \frac{\partial F}{\partial \sigma_{ij}}$

   ここで、
   - $\dot{\varepsilon}_{ij}^p$: 塑性ひずみ速度
   - $\lambda$: 正の比例定数（塑性乗数）
   - $F$: 降伏関数（塑性ポテンシャル）

### 4. 特徴:
   - 塑性ひずみ増分ベクトルは降伏曲面に垂直な方向を向く
   - ミーゼスの降伏条件を用いた場合、体積一定の条件を満たす

### 5. 降伏状態の判定:
   - $\frac{\partial F}{\partial \sigma_{ij}} d\sigma_{ij} < 0, F = 0$: 弾性除荷
   - $\frac{\partial F}{\partial \sigma_{ij}} d\sigma_{ij} = 0, F = 0$: 中立負荷
   - $\frac{\partial F}{\partial \sigma_{ij}} d\sigma_{ij} > 0, F = 0$: 塑性負荷

### 6. 注意点:
   - $\frac{\partial F}{\partial \sigma_{ij}}$ は降伏曲面の法線方向を表す
   - 応力増分について、(1)降伏曲面の外側に向かう場合は塑性負荷、(2)降伏曲面の接線方向を向いている場合は中立負荷、(3)降伏曲面の内側を向いている場合は弾性除荷となる

# 2-13

## 負荷と除荷、流れ則

### 1. 降伏関数:
   $F = F(\sigma_{ij}, a, b_i, c_{ij})$

   ここで、$a, b_i, c_{ij}$ はスカラー、ベクトル、テンソル量のパラメータで、硬化係数や材料定数を表す。

### 2. 降伏条件:
   $F = F(\sigma_{ij}, a, b_i, c_{ij}) = 0$

### 3. 全微分:
   $dF = \frac{\partial F}{\partial \sigma_{ij}} d\sigma_{ij} + \frac{\partial F}{\partial a} da + \frac{\partial F}{\partial b_i} db_i + \frac{\partial F}{\partial c_{ij}} dc_{ij}$

### 4. 負荷条件:
   $F = 0$ かつ $dF < 0$ であると、塑性状態が進行する。

### 5. 流れ則:
   $d\varepsilon_{ij}^p = d\lambda \frac{\partial F}{\partial \sigma_{ij}}$

   ここで、$d\varepsilon_{ij}^p$ は塑性ひずみ増分、$d\lambda$ は塑性乗数の増分。

### 6. 特徴:
   - 塑性ひずみ増分ベクトルは降伏曲面の外向き法線方向に一致
   - これを関連条件（または会合則）と呼ぶ

### 7. 加工硬化する材料の場合:
   塑性変形が進行中は $d\varepsilon_{ij}^p$ は降伏曲面の外向き法線方向となる。このとき応力増分 $d\sigma_{ij}$ は $d\varepsilon_{ij}^p$ と同じ向きを取る。

# 2-14

## 相当応力、相当ひずみ

### 1. ミーゼスの条件による相当応力:
   偏差応力テンソルの第2不変量に関連したもの

### 2. 特徴:
   - 主応力を用いて簡単に表現可能
   - 応力から容易に平均応力を除いた値
   - 偏差応力が得られる

### 3. 相当ひずみ:
   - 多軸応力状態を適当な一次元の尺度で表したもの
   - 塑性仕事の概念から誘導される
   - 単軸引張試験の塑性ひずみと等価なもの

### 4. 重要性:
   - 複雑な応力状態を一つの値で表現できる
   - 材料の降伏や硬化挙動を簡単に評価できる

### 5. 応用:
   - 多軸応力状態における材料の挙動予測
   - 塑性変形の程度を定量化する際に使用
   - 降伏条件や硬化則の定式化に利用

### 6. 注意点:
   - 相当応力と相当ひずみは、必ずしも実際の応力やひずみを直接表すものではない
   - 材料や負荷条件によっては、より複雑なモデルが必要な場合がある

# 2-15

## 相当応力の計算

### 1. 計算方法:
相当応力 $\bar{\sigma}$ は偏差応力テンソル $s_{ij}$ を用いて次のように計算される：
$\bar{\sigma} = (\frac{3}{2}s_{ij}s_{ij})^{1/2}$

### 2. 具体的な計算例:
(a) 応力と偏差応力が以下の場合：
$[\sigma] = \begin{bmatrix} 200 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{bmatrix}$, 
$[s] = \begin{bmatrix} 400/3 & 0 & 0 \\ 0 & -200/3 & 0 \\ 0 & 0 & -200/3 \end{bmatrix}$

相当応力は $\bar{\sigma} = 200$ となる。

(b) 応力と偏差応力が以下の場合：
$[\sigma] = [s] = \begin{bmatrix} 100 & 0 & 0 \\ 0 & -100 & 0 \\ 0 & 0 & 0 \end{bmatrix}$

相当応力は $\bar{\sigma} = 100\sqrt{3}$ となる。

(c) 応力と偏差応力が以下の場合：
$[\sigma] = [s] = \begin{bmatrix} 0 & 100 & 0 \\ 100 & 0 & 0 \\ 0 & 0 & 0 \end{bmatrix}$

相当応力は $\bar{\sigma} = 100\sqrt{3}$ となる。
$\sigma_{(a)} > \sigma_{(b)} = \sigma_{(c)}$ となるので、(a)が正解である。

# 2-16

## 弾塑性解析と剛塑性解析

### 1. 弾塑性解析:
- 弾性変形と塑性変形の両方を考慮
- 弾性ひずみが塑性ひずみに比べて小さい場合、これを無視することがある
- 剛塑性体(rigid-plastic material)として扱う場合もある

### 2. 剛塑性解析の特徴:
- 弾性ひずみがないため、弾性変形を伴う除荷を扱うことができない
- 通常の変形では塑性変形は体積一定の変形（非圧縮性）であるので、要素の連続に注意する必要がある

### 3. 解析手法:
- 一般に剛塑性有限要素法では、レヴィ-ミーゼスの式（Levy-Mises equation）を用いて上界定理（upper bound theorem）に基づいた近似解が得られる

### 4. 剛塑性解析の利点:
- 弾性変形を無視することで、塑性変形のみに注目できる
- 計算が比較的容易

### 5. 剛塑性解析の制限:
- 弾性変形や除荷過程を考慮できない
- 変形の初期段階や、弾性変形が重要な場合には適さない

### 6. 応用:
- 金属加工プロセスの解析
- 大変形問題の解析

### 7. グラフ解釈:
- (a) 完全塑性体：降伏後、応力が一定のまま変形が進む
- (b) 剛完全塑性体：降伏まで変形せず、降伏後に塑性変形が進む

# 2-17

## エネルギー原理

### 1. 概要:
弾性体ではポテンシャルエネルギーの原理が成り立つが、剛塑性体では成立しない。

### 2. 弾性体の場合:
- 系のポテンシャルエネルギーが最小になる変位場が真の解となる
- 全ポテンシャルエネルギー最小の原理（total potential energy minimum principle）が成立する

### 3. 剛塑性体の場合:
- エネルギー原理として、上界定理（upper bound theorem）および下界定理（lower bound theorem）が成立する
- これらの定理は塑性極限解析の基礎となっている

### 4. 上界定理:
- 運動学的に可能な速度場を仮定し、その速度場に対する塑性仕事率が外力仕事率以上であれば、その外力は真の崩壊荷重以上である

### 5. 下界定理:
- 静力学的に可能な応力場を仮定し、その応力場が降伏条件を満たせば、その外力は真の崩壊荷重以下である

### 6. 応用:
- これらの定理を用いることにより剛塑性体の解析が可能となる
- 極限解析や崩壊荷重の評価に広く用いられる

# 2-18

## エネルギー原理

### 1. 概要:
剛塑性体に対するエネルギー原理（上界定理、upper bound theorem）は、与えられた境界条件の下で全エネルギー散逸率を最小にする運動学的に可能な速度場を考えるという原理である。

### 2. 弾塑性体との比較:
弾性体に対する最小ポテンシャルエネルギーの原理に対応している。

### 3. 剛塑性体の特徴:
以下の対応関係が成り立つ：

| 弾性体 | 剛塑性体 |
|--------|----------|
| 応力 | 応力 |
| ひずみ | ひずみ速度 |
| 力 | 力 |
| 変位 | 変位速度 |

### 4. エネルギー関係:
剛塑性体では応力はひずみ速度に依存しないと仮定するため、

$\int \sigma_i d\varepsilon_i = \frac{1}{2}\varepsilon_{ij}\sigma_{ij}$

となり、非線形な最小化問題となる。

### 5. 応用:
- 塑性極限解析
- 崩壊荷重の評価
- 金属加工プロセスの解析

### 6. 注意点:
- この原理は剛塑性体に対してのみ適用可能
- 実際の材料挙動とは差異が生じる可能性がある

# 2-19

## シェイクダウン

### 1. 定義:
構造物が繰返し負荷を受ける場合について解析することを極限解析（limit analysis）という。

### 2. シェイクダウンの概念:
多くの場合、繰り返し負荷は単純な繰返し塑性体として取り扱われる。構造全体としての変形が、ある一定の繰返し数の後に弾性的挙動を示すようになることをシェイクダウンという。

### 3. シェイクダウンの過程:
1. 最初に塑性変形が生じる
2. 繰返し数の増加とともに変形が蓄積
3. やがて弾性範囲内の挙動に収束

### 4. シェイクダウン定理:
- メランの定理（Melan's theorem, 静力学定理）
- コイターの定理（Koiter's theorem, 運動学定理）

これらは塑性設計の基礎となる重要な定理である。

### 5. インクリメンタルコラプス:
シェイクダウンせずに、繰返し負荷とともに変形が単調に増加し続ける現象をインクリメンタルコラプス（incremental collapse）という。

### 6. 応用:
- 構造物の疲労設計
- 繰返し荷重を受ける機械部品の設計
- 原子力プラントの安全性評価

### 7. 重要性:
シェイクダウン解析は、構造物が繰返し負荷に対して長期的に安全であるかを評価する上で極めて重要な手法である。

### 8. 注意点:
- 実際の材料挙動は理想的なシェイクダウン挙動から乖離する可能性がある
- 複雑な負荷履歴や材料の非線形性を考慮する場合、より高度な解析手法が必要となる場合がある

# 問2-20 塑性曲げ、残留応力

## 塑性曲げと残留応力

### 1. 長方形断面梁の塑性モーメント:
   断面の高さ h の中央部（すなわち中立軸）で
   M = P/4 となる。

### 2. 塑性曲げの特徴:
   h/2 の断面がすべて塑性変形する場合には、応力分布は：
   σ = M / I

### 3. 断面二次モーメント:
   I = bh^3/12（b: 幅, h: 高さ）

### 4. 応力分布:
   σ = y / I * -π < y < η
   ここで、I は断面二次モーメント、y は中立軸からの距離。

### 5. 残留応力:
   曲げ応力が除荷されると、残留応力が発生する。これは弾性限界を超えた塑性変形によるものです。

### 6. 応力分布の特徴:
   残留応力は断面内で自己平衡状態となる。つまり、断面全体での応力の総和はゼロになります。

### 7. 注意点:
   - 塑性変形が進むにつれて、中立軸の位置が変化する可能性がある。
   - 残留応力の分布は、負荷履歴や材料特性に依存する。

### 8. 応用:
   この知識は、構造物の設計や強度評価、特に過負荷後の構造物の挙動予測に重要です。

# 2-21

## 熱応力による降伏

### 1. 熱応力の定義:
温度が変化すれば材料は変形（一般に膨張・収縮）し、この自由な変形を拘束すれば応力が生じる。この応力を熱応力（thermal stress）という。

### 2. 熱応力の発生メカニズム:
- 材料が等方で線膨張係数（thermal expansion coefficient）α が温度について一定であると仮定
- 温度変化が ΔT [℃] によって生じるひずみ（熱ひずみ, thermal strain）は ε = αΔT

### 3. 計算例:
- 鋼材の場合: α = 10×10^(-6) /℃, E = 200000MPa
- 温度上昇 T = 300℃ の場合:
  σ = 300 - 0.3×T = 200000×10×10^(-6)×T = 2×T
- 結果: T = 130℃ で降伏

### 4. 注意点:
- この計算は鋼の値に近いものであって、実際の値とは異なる
- 熱応力の効果は一般に小さくないことを示している

ここで、

- 材料の非線形性（塑性変形）と温度依存性を同時に扱う

## 熱弾塑性解析

### 1. 目的:
温度変化に伴って生じた熱ひずみを、応力解析に取り入れること。

### 2. 手順:
1. 全ひずみ {ε} を求める
2. 熱ひずみ {ε^T} を差し引く
3. 応力 {σ} を計算する

### 3. 数式表現:
$\{ε\} = [D]^{-1}\{\{σ\} - \{T\}\}$

ここで、{T} は熱応力ベクトル。

### 4. 全ひずみの分解:
$\{ε\} = \{ε^e\} + \{ε^p\} + \{ε^T\}$

ここで、
- {ε^e}: 弾性ひずみ
- {ε^p}: 塑性ひずみ
- {ε^T}: 熱ひずみ

### 5. 解析手順:
弾性によって生じるひずみの値は既知で、
$\{ε^e\} = [D]^{-1}\{σ\} + [D]^{-1}\{T\} + [D]^{-1}[D]\{ε^p\}$

# 問2-25 クリープ構成式

## クリープ現象に関する構成式

### 1. クリープ構成式の一般形：
   $\dot{\varepsilon}^c = f(\sigma, t, T)$
   ここで、$\dot{\varepsilon}^c$はクリープひずみ速度、$\sigma$は応力、$t$は時間、$T$は温度を表す。

### 2. 第2期クリープのクリープ構成式（ノルトン則）：
   $\dot{\varepsilon}^c = k\sigma^n$
   ここで、$k, n$は温度の関数。

### 3. ブラックバーン型の構成式：
   $\dot{\varepsilon}^c = A(1-\exp(-\alpha t)) + B(1-\exp(-\alpha t)) + Ct$
   ここで、$A, B, C, r, s$は応力と温度の関数。
   第1項は第1期クリープ、第2項は第2期クリープ、第3項は第3期クリープを表している。

### 4. クリープ曲線を解析する2つの方法：
   a. ホフ（Hoff）の解析を用いて定常クリープ状態（時間とともに応力が変化しない状態）を求める方法。
   b. クリープにおける定常状態と非定常弾性変形状態の類似性を利用する方法。

### 5. 定常状態におけるクリープひずみ速度：
   $\dot{\varepsilon}^c = k\sigma^n$

### 6. 非線形粘性体の解析：
   応力状態が時間とともに変化する非定常クリープを解析するために、弾塑性変形と同様に、逐次変形を追跡する解析を行う。

### 7. クリープ硬化則の適用：
   応力変動を伴う非定常クリープ解析で第1期クリープを含む構成式を用いる場合には、クリープ硬化則を用いる必要がある。

### 8. 注意点：
   - 定常クリープ状態を求める場合でも、実際には一番目の方法ではなく二番目の非定常クリープ解析の結果として定常クリープ状態を求める方法が多く用いられている。

# 2-27

## 粘弾性体の力学モデル

### 1. 概要:
粘弾性材料、コンクリートなどは弾性的性質と粘性的性質を合わせ持つ材料特性を示す。

### 2. 粘弾性挙動の特徴:
- 弾性ばねと粘性ダッシュポットの組み合わせによりモデル化できる。

### 3. 代表的なモデル:
1. フォークトモデル (Voigt model)
2. マックスウェルモデル (Maxwell model)

### 4. フォークトモデル:
- 弾性ばね $E$ とダッシュポット $\eta$ が並列に接続
- 応力とひずみの関係: $\sigma = E\varepsilon + \eta\frac{d\varepsilon}{dt}$
- 特徴: ばねとダッシュポットのひずみは共通であり、応力はばねの力とダッシュポットの力の和

### 5. マックスウェルモデル:
- 弾性ばね $E$ とダッシュポット $\eta$ が直列に接続
- ひずみ速度の式: $\frac{d\varepsilon}{dt} = \frac{1}{E}\frac{d\sigma}{dt} + \frac{\sigma}{\eta}$
- 特徴: ばねとダッシュポットの応力は共通であり、全体のひずみはばねのひずみとダッシュポットのひずみの和

### 6. クリープ挙動の表現:
フォークトモデルに時間 $t=0$ で応力 $\sigma_0$ を与えた場合のひずみ応答:
$\varepsilon = \frac{\sigma_0}{E}[1 - \exp(-t/T_r)]$
ここで、$T_r = \eta/E$ は遅延時間である。

### 7. 応力緩和の表現:
マックスウェルモデルに初期ひずみ $\varepsilon_0$ を与えた場合の応力応答:
$\sigma = E\varepsilon_0\exp(-t/T_r)$
ここで、$T_r = \eta/E$ は緩和時間である。

図2には、(a)クリープ挙動と(b)応力緩和挙動のグラフが示されている。

    - 大変形を伴う構造解析

# 2-29 超弾性体

## 1. 定義と特徴
- 超弾性体（hyperelastic）は、エネルギー密度がひずみの解析関数で表すことができるもの。
- 厳密に自然状態と変形状態に1対1の関係をもつ。
- 応力はエネルギー密度関数をポテンシャルとして導かれる。

## 2. 応力とひずみの関係
- 応力とひずみ速度の線形関係は、亜弾性体（hypoelastic）と呼ばれる。
- 超弾性体も亜弾性体の特殊な場合である。
- 応力とひずみ速度の関係は、粘性材料の一般的な構成式を表しており、ニュートン流体の特殊な場合に相当する。

## 3. 大変形の取り扱い
- ゴムのような大きな弾性変形を扱う場合には、応力やひずみの定義を明確にした解析手法を用いることが必要。

## 4. エネルギー密度関数
- 超弾性体の挙動は、エネルギー密度関数（ポテンシャル）によって記述される。
- このポテンシャルから応力を導くことができる。

## 5. 実験的側面
- 超弾性体の特性を正確に把握するには実験が必要。
- 例えば、100%のひずみ領域までの実験で求めた弾性ポテンシャル関数を用いて、150%のひずみ領域を計算した場合、100%以上の結果は実際と大きく異なることが多い。

## 6. 応用
- ゴムや弾性体の大変形解析に広く用いられる。
- 生体組織のモデル化にも適用される場合がある。

## 7. 注意点
- 大変形を扱う際には、ひずみの定義や応力の定義に注意が必要。
- 実験データの範囲を超えた外挿には注意が必要。

## 8. 他の弾性モデルとの関係
- 小さな変形の範囲では線形弾性体と同様の挙動を示す。
- 大きな変形領域では非線形性が顕著になる。
