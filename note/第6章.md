# 6-1

## 動的解析における運動方程式

### 1. 運動方程式の基本形:
   時間的に変動する外力に対する構造物の応答を表す運動方程式は以下の形で表される：

   $[M]\{\ddot{u}\} + [C]\{\dot{u}\} + [K]\{u\} = \{f\}$

   ここで、
   - $[K]$: 剛性マトリックス
   - $[M]$: 質量マトリックス
   - $[C]$: 減衰マトリックス
   - $\{\ddot{u}\}$: 加速度ベクトル
   - $\{\dot{u}\}$: 速度ベクトル
   - $\{u\}$: 変位ベクトル
   - $\{f\}$: 荷重ベクトル

### 2. 有限要素法による解析:
   - 左辺第1項から式(*)右辺が導かれる
   - 線形有限要素法の場合、式(**)左辺第2項から式(*)左辺第3項が得られる
   - 非線形を考慮する場合、式(*)にニュートン・ラプソン法を適用し、静的解析で用いられる接線剛性マトリックスを置き換えるなどの操作が必要

### 3. 仮想仕事の原理:
   式(*)左辺第2項は仮想仕事の原理には直接対応する項はないが、物体の運動により未知の速度に比例する外力が作用すると考え導入されるもの

### 4. 減衰マトリックス:
   - 形式が定まっているものではない
   - 例えば問6-6解説に示すようなレイリー減衰など何らかの形式が仮定されることが多い

- 密度の平方根に反比例する

# 6-3

## 弾性波の種類と特徴

### 1. 弾性波の分類:
   固体内弾性波には以下の2種類がある：
   - P波（Primary Wave または Pressure Wave）
   - S波（Secondary Wave または Shear Wave）

### 2. P波の特徴:
   - 波の進行方向と振動方向が同じ
   - 縦波または疎密波とも呼ばれる
   - 速度は岩盤中で5から7 km/sec
   - 固体・液体・気体中を伝播可能

### 3. S波の特徴:
   - 波の進行方向と振動方向が垂直
   - せん断波とも呼ばれる
   - 速度は岩盤中で3から4 km/sec
   - 固体内のみに存在する

### 4. 速度の比較:
   P波の速度はS波の速度より速い

### 5. 重要な違い:
   S波の振動方向は波の進行方向と垂直であるのに対し、P波は波の進行方向と振動方向が同じである

# 6-4

## 質量マトリックス

### 1. 質量マトリックスの定義:
   運動方程式を離散化すると、要素集合質量マトリックス[m]は以下の式で表される：

   $[m] = \int_v \{N\}^T \rho \{N\} dv$

   ここで、
   - $\{N\}$: 形状関数ベクトル
   - $\rho$: 密度
   - $v$: 体積

### 2. 梁要素の場合:
   $\{N\} = \{N_1, N_2\}$

   $[m] = \int_0^l \begin{bmatrix} N_1 \\ N_2 \end{bmatrix} \rho [N_1, N_2] dx$

### 3. 要素の体積積分:
   $\int_v dv = A \int_0^l dx$

   ここで、A は断面積

### 4. 質量マトリックスの計算:
   $m_{11} = \frac{1}{3} \rho AL$
   $m_{12} = m_{21} = \frac{1}{6} \rho AL$
   $m_{22} = \frac{1}{3} \rho AL$

### 5. 最終的な質量マトリックス:
   $[m] = \frac{\rho AL}{6} \begin{bmatrix} 2 & 1 \\ 1 & 2 \end{bmatrix}$

# 6-5

## 剛性マトリックス

### 1. 剛性マトリックスの定義:
   要素剛性マトリックス $[k^e]$ は以下のように表される：

   $[k^e] = \int_v [B]^T [D] [B] dv$

   ここで、
   - $[B]$: ひずみ-変位マトリックス
   - $[D]$: 応力-ひずみマトリックス
   - $v$: 体積

### 2. 質量マトリックスとの関係:
   $[m^e] = \int_v \rho [N]^T [N] dv$

### 3. マトリックスの構造:
   $[N] = \begin{bmatrix}
   N_1 & 0 & 0 & N_2 & 0 & 0 \\
   0 & N_1 & 0 & 0 & N_2 & 0 \\
   0 & 0 & N_1 & 0 & 0 & N_2
   \end{bmatrix}$

   $[B] = \begin{bmatrix}
   \frac{\partial}{\partial x} & 0 & 0 \\
   0 & \frac{\partial}{\partial y} & 0 \\
   0 & 0 & \frac{\partial}{\partial z} \\
   \frac{\partial}{\partial y} & \frac{\partial}{\partial x} & 0 \\
   0 & \frac{\partial}{\partial z} & \frac{\partial}{\partial y} \\
   \frac{\partial}{\partial z} & 0 & \frac{\partial}{\partial x}
   \end{bmatrix} [N]$

# 6-6

## 減衰マトリックス

### 1. 減衰マトリックスの定義:
   構造物の減衰は外部摩擦（空気、水）の粘性に起因する減衰や、構造物の内部摩擦などに起因する減衰があり、これらを減衰マトリックス[C]として表現する。

### 2. レイリー減衰:
   $[C] = a[M] + b[K]$

   ここで、
   - $[M]$: 質量マトリックス
   - $[K]$: 剛性マトリックス
   - $a, b$: 比例定数

### 3. 減衰比の計算:
   $\zeta = \frac{a}{2\omega} + \frac{b\omega}{2}$

   ここで、$\omega$ は固有角振動数

### 4. 比例定数の決定:
   2つの固有振動数 $\omega_1, \omega_2$ に対する減衰比 $\zeta_1, \zeta_2$ を与えて、$a, b$ を決定する。

   $a = 2\omega_1\omega_2 \frac{(\zeta_2\omega_1 - \zeta_1\omega_2)}{\omega_1^2 - \omega_2^2}$

   $b = 2 \frac{(\zeta_1\omega_1 - \zeta_2\omega_2)}{\omega_1^2 - \omega_2^2}$

### 5. 特徴:
   - 低周波数領域では質量比例型減衰が支配的
   - 高周波数領域では剛性比例型減衰が支配的
   - 多自由度系の場合、着目する周波数帯域を定め、上限を $\omega_2$、それぞれの減衰比を $\zeta_1, \zeta_2$ とすると、$a, b$ の値を求めることができる

### 6. 応用:
   レイリー減衰は質量マトリックスと剛性マトリックスの線形結合で表されるため、プログラミングが容易であり、モード解析でも用いることができる。

# 6-7

## アワーグラスコントロール

### 1. 目的:
   直接時間積分法による動的解析において、剛解法で生じる振動を抑制するために時間増分を設定する。

### 2. 特徴:
   - 一般にこの時間増分は微小な値となるため、計算サイクル時間との群遅延時間を小さくするように、低減積分要素を用いることが推奨されている。
   - 二次元問題の場合、低減積分要素を用いた場合の時間増分は、完全積分要素を用いた場合に比べて一般に数分の1程度になるとされている。

### 3. アワーグラスモード:
   - 低減積分要素を用いると、アワーグラスモード（hour glass mode、問8-18、問8-19の解説参照）が誘発されることがあるため、アワーグラスモードの発生抑制のための仮想的な力（アワーグラス抗力）を導入する。
   - 多くの場合、アワーグラスモードが生じても計算を継続できるので、得られた計算結果を詳細に観察する必要がある。

### 4. 注意点:
   - アワーグラスの発生は、計算精度を著しく低下させるため、考えなければならない。
   - マスマトリーリング（問6-16解説参照）とアワーグラスモードの発生は関係ないので間違い。
   - アワーグラスエネルギーは、アワーグラス変位とアワーグラス抗力の積の累積値として評価される。
   - アワーグラス抗力は一般にアワーグラス変位に比例する形で与えられるため、アワーグラス変位が生じているとアワーグラス抗力も大きくなり、その結果アワーグラスエネルギーも大きくなる。

# 6-8

## 衝撃計算におけるエネルギー

衝撃解析で自動車の衝突解析を実施すると、解析結果の観点から、1点積分の低次要素を使用することが多い。また1点積分の低次要素を使用するとアワーグラスモード（問8-18、問8-19の解説参照）で模擬部の質入れ押し抜すような問題（問4-6解説参照）が発生するため、低周波数ノイズの抑制のために仮想的な力（アワーグラス抗力）を導入する。（問6-7解説参照）

アワーグラスモードを抑制するために使用されるアワーグラス変位を時間積分して求める。非物理的な力であるため、（問6-7解説参照）

解析を行う場合、解析結果の検証のために運動エネルギー、内部エネルギー、外部仕事、アワーグラスエネルギー、摩擦エネルギーの和である全エネルギーを時間積分においてほぼ一定であること、不連続な変化がないことを確認することが望ましい。自動車の衝突解析では、最初に全運動エネルギーであり、変形が進行し、最後には全てのエネルギーは内部エネルギーになる。要素が大きく変形すると、運動エネルギー全てが変形によるひずみエネルギーに変換される。衝突解析においては、エネルギーの流れを追跡することにより、妥当な挙動をしているか確認することができる。またアワーグラスエネルギーも確認して妥当かどうか確認することが望ましい。接触エネルギーは、買入と反発力（問4-6解説参照）で表され、自動車の衝突解析では重要なパラメータと考えられている。なお、衝突解析はされた位置エネルギーも加わる。

解析解では力が作用しないので全体的なエネルギーは一定となる。

# 6-9

## 中心差分法

外力が構造物に作用すると構造物には運動と変形が発生する。運動する構造物の運動方程式を考えると、有限要素法により空間離散化において離散化した運動方程式は次のようになる。

$[M]\{\ddot{u}\}+[C]\{\dot{u}\}+[K]\{u\}=\{f\}$

この運動方程式は二階の常微分方程式であるため、解を求めるには時間に関する離散化を行う必要がある。中心差分法、ニューマーク β 法などの時間に関する離散化方法があり、以下の方法で解が得られることを示す。

時刻 $t_{n+1}$ における変位ベクトル $\{u\}_{n+1}$ について次のテイラー展開式を考える：

$\{u\}_{n+1} = \{u\}_n + \Delta t\{\dot{u}\}_n + \frac{(\Delta t)^2}{2!}\{\ddot{u}\}_n + \cdots$

$\{u\}_{n-1} = \{u\}_n - \Delta t\{\dot{u}\}_n + \frac{(\Delta t)^2}{2!}\{\ddot{u}\}_n - \cdots$

(1)式と(2)式の和をとると：

$\{\dot{u}\}_n = \frac{\{u\}_{n+1} - \{u\}_{n-1}}{2\Delta t}$

(1)式と(2)式の差をとると：

$\{\ddot{u}\}_n = \frac{\{u\}_{n+1} - 2\{u\}_n + \{u\}_{n-1}}{(\Delta t)^2}$

(3)式および(4)式を時間 $t_n$ における運動方程式に代入すると：

$[M]\{\ddot{u}\}_n + [C]\{\dot{u}\}_n + [K]\{u\}_n = \{f\}_n$

$\frac{1}{(\Delta t)^2}[M](\{u\}_{n+1} - 2\{u\}_n + \{u\}_{n-1}) + \frac{1}{2\Delta t}[C](\{u\}_{n+1} - \{u\}_{n-1}) + [K]\{u\}_n = \{f\}_n$

$= (\{f\}_n - [K] - \frac{2}{(\Delta t)^2}[M])\{u\}_n - (\frac{1}{(\Delta t)^2}[M] - \frac{1}{2\Delta t}[C])\{u\}_{n-1}$

# 6-10 ニューマーク β 法

1. 特徴：
   - γ, β は任意の値を選ぶことができる
   - 精度および安定性を考慮して設定される
   - 一般的に γ = 1/2, β ≥ 1/4 で無条件安定となる
   - γ = 1/2, β = 1/4 が用いられることが多い

2. 時間増分と加速度変化：
   - β = 1/6 のとき、時間増分内で加速度は直線的に変化し、「線形加速度法」と呼ばれる
   - β = 1/4 のとき、時間増分内で加速度は一定となり、「平均加速度法」と呼ばれる

3. 計算手順：
   - 運動方程式と初期条件から出発
   - ニューマーク β 法の近似式を用いて変位と速度を計算
   - γ = 1/2, β = 1/4 を代入して具体的な計算式を導出

4. 特性：
   - 平均加速度法の場合、時間領域の離散化による数値減衰が生じない
   - 無条件安定であり、時間増分の取り方にかかわらず安定な解が得られる

---

# 6-11 HHT-α 法

1. 目的：
   - 高次モードに対する数値減衰を導入しつつ、低次モードの精度を保つ

2. 特徴：
   - パラメータ α（一般的に α = -0.1~-0.5 の値）を導入
   - ニューマーク β 法を拡張した形式となっている

3. 計算式：
   $[M]\{^{t+\Delta t}\ddot{u}\} + [C][(1+\alpha)\{^{t+\Delta t}\dot{u}\}-\alpha\{^t\dot{u}\}] + [K][(1+\alpha)\{^{t+\Delta t}u\}-\alpha\{^tu\}] = (1+\alpha)\{^{t+\Delta t}f\}-\alpha\{^tf\}$

4. 性質：
   - α = 0 のとき、ニューマーク β 法に帰着する
   - 高次モードに対する数値減衰を制御しやすい
   - 低次モードの精度を保ちつつ、高次モードの影響を抑制できる

5. 適用：
   - 構造物の地震応答解析など、高周波数成分を含む問題に有効

---

# 6-12 陰解法・陽解法

1. 運動方程式：
   $[M]\{\ddot{u}\} + [C]\{\dot{u}\} + [K]\{u\} = \{f\}$
   
   ここで、[M]は質量マトリックス、[C]は減衰マトリックス、[K]は剛性マトリックス、{u}は変位ベクトル、{f}は外力ベクトルを表す。

2. 時刻t+Δtでの運動方程式：
   $[M]\{\ddot{u}\}_{t+\Delta t} + [C]\{\dot{u}\}_{t+\Delta t} + [K]\{u\}_{t+\Delta t} = \{f\}_{t+\Delta t}$

3. 解法の分類：
   a) 陽解法：時刻tの運動方程式を用いて{u}t+Δtを求める方法
   b) 陰解法：時刻t+Δtの運動方程式を用いて{u}t+Δtを求める方法
   c) その他：aとbの解を近似的に評価する方法

4. 陰解法の特徴：
   - ニューマークβ法が代表的な例
   - 無条件安定であり、大きな時間ステップを使用可能
   - 各ステップで連立方程式を解く必要がある

5. 陽解法の特徴：
   - 中心差分法が代表的な例
   - 条件付き安定であり、時間ステップに制限がある
   - 各ステップでの計算が簡単

6. 計算式：
   陰解法（質量マトリックスが対角化されている場合）：
   $\frac{1}{(\Delta t)^2}[M]\{^{t+\Delta t}u\} = \{f\}^*$
   
   ここで、{f}*は既知の値を含む右辺項

7. 解法の選択：
   問題の性質や要求される精度、計算効率などを考慮して適切な解法を選択する必要がある。

# 6-13 安定条件

1. 安定性の定義：
   数値解法において、微小な誤差が時間の経過とともに増幅されないこと。

2. 陽解法の安定条件：
   陽解法では一般に安定条件が存在し、時間増分Δtに上限がある。

3. クーラン条件：
   $c = \frac{v \cdot \Delta t}{l} \leq 1$
   
   ここで、vは波動の伝播速度、lは要素長さを表す。

4. 一般的な材料での安定条件：
   $\Delta t \leq \frac{l}{c_p}$
   
   ここで、cpは縦波（P波）の伝播速度を表す。

5. 金属材料の場合：
   縦波の伝播速度が大きいため、時間増分が非常に小さくなる。

6. 解決策：
   - マススケーリング：質量を人工的に大きくすることで安定限界を上げる
   - 陰解法の使用：無条件安定であるため、大きな時間ステップが使用可能

7. 注意点：
   安定条件を満たしていても、必ずしも精度の良い解が得られるとは限らない。問題の性質に応じて適切な時間増分を選択する必要がある。

---

# 6-14 安定条件

1. 波動の伝播速度：
   - 密度をρ、応力をσとすると、波動の伝播速度cは $c = \sqrt{\frac{d\sigma}{d\varepsilon}}$ で表される
   - 一般の金属材料では弾性波の伝播速度が非常に大きい

2. 時間増分の決定：
   - 波動の伝播速度が大きい場合、時間増分を非常に小さく設定する必要がある
   - 金属以外の材料でも、要素に伴って要素形状が小さくなり、しかも応力-ひずみ関係のこう配が大きくなるような場合は注意が必要

3. 自動時間増分設定：
   - 汎用の衝撃解析ソフトウェアでは自動的に時間増分設定を修正してクーラン条件を満たすようになっている
   - クーラン条件：$c = \frac{v \cdot \Delta t}{l} \leq 1$（vは波動の伝播速度、lは要素長さ）

# 6-15 マルチタイムステップ法

1. 目的：
   - 安定かつ効率的な動解析を行うための手法

2. 特徴：
   - 解析領域を分割し、各領域に適した時間増分を用いる
   - 剛性の制限や解析メッシュの密度などにより、使用するメッシュに含まれる要素のサイズに大きな差がある場合に有効

3. メリット：
   - 解析に要する計算時間を大幅に短縮できる

4. 適用例：
   - サブサイクリング法として知られている

5. 注意点：
   - 1時間ステップ当たりの計算量が少なくなる反面、時間増分を大きくすると解が発散しやすくなる
   - 安定性と精度を保つために適切な時間増分の設定が必要

---

# 6-16 動的緩和法

1. 概要：
   - 複雑な非弾性挙動を伴う構造の静的解析を行う際に用いられる手法
   - 強制振動の影響を受けず、運動増分法を用いて静的つり合い状態を求める

2. 特徴：
   - すべての節点の速度、加速度を繰り返し計算して静的な釣り合い状態を求める
   - 動的緩和法では静的な釣り合い状態のみを求めるため、中間状態は物理的な意味を持たない

3. 適用：
   - 幾何学的非線形性を考慮した解析
   - 材料非線形性を考慮した解析
   - 接触を考慮した解析

4. 注意点：
   - 静的解析で用いる場合、静的つり合い状態を求めるだけでもクーラン条件（問6-13解説参照）の制約を受けるため、計算時間が長くなる可能性がある

5. 解決策：
   - 質量の人工的な付加によって時間ステップを大きくする手法（マススケーリング）が用いられる
   - ただし、慣性項エネルギーなど衝撃解析に比べて運動の影響が小さくなるように注意が必要

# 6-17 モード解析

1. 目的：
   - 固有モードの抽出と動解析での活用
   - 複雑な系の振動特性を簡単に把握する

2. 特徴：
   - 固有モードの数は有限要素法により離散化されたモデルでは自由度に等しい
   - すべての固有ベクトルの線形和として表すことができる
   - 代表的な固有ベクトルのみを選択することができる

3. 利点：
   - 精度は落ちるが、計算時間の短縮が可能
   - コンピュータリソースの節約ができる

4. 計算方法：
   運動方程式を固有値問題として解く
   $MK + Kx = 0$

   ここで、
   $M = \begin{bmatrix} m_1 & 0 \\ 0 & m_2 \end{bmatrix}$
   $K = \begin{bmatrix} k_1+k_2 & -k_2 \\ -k_2 & k_2 \end{bmatrix}$

5. 解の導出：
   $det|-λ^2M + K| = 0$

   $det\begin{vmatrix} -λ^2m_1 + k_1 + k_2 & -k_2 \\ -k_2 & -λ^2m_2 + k_2 \end{vmatrix} = 0$

6. 結果：
   固有角振動数と固有ベクトルが得られる

# 6-18 モード解析法

1. 目的：
   振動現象の解析に注意すべき点を理解し、「直接時間積分法」と「モード解析法」の違いを理解する

2. 特徴：
   - 固有モードと固有ベクトルからなり、固有値は固有振動数とも呼ばれる
   - 固有ベクトルは固有モードとも呼ばれる

3. 解析方法：
   (1) 固有値解析では、しばしば反復解法の形で、剛直度と質量に反復的に修正を加えることが主流
   (2) 固有値は sin x の形で表現される
   (3) 運動方程式を固分式で表現すると、MK + Kx の形になる

4. 重要性：
   - 固有ベクトルの大きさは任意に定めることができる
   - 解析結果として示す場合には固有ベクトルの大きさを1とするようにするなどである

# 6-19 モード解析法・応答スペクトル法

1. 目的：
   地震時構造物の動的応答を評価する方法

2. 手法：
   - 時刻歴応答解析（time-history analysis）
   - モード重ね合わせによる時刻歴応答解析
   - モード解析法（modal analysis）
   - 応答スペクトル法（response spectrum approach）

3. 特徴：
   - モード解析と応答スペクトル法は、非線形自由度の少ないモードに対して適用可能
   - 基本的には、質量マトリックスが対角マトリックスで表現される集中質量系に適用される

4. 適用条件：
   - 直接時間積分法は非線形性が強い場合に有効
   - 構造物の自由度が大きい場合は、計算コストは極めて大きくなる

5. 解析手法の選択：
   - ニューマーク（Newmark）β法では、β≧1/4のとき無条件安定となる
   - 高次モードの影響を抑制するために数値減衰を導入する場合もある
   - 非線形性が強い場合は、直接時間積分法を用いる必要がある

6. 注意点：
   - 高周波数領域では数値減衰が過大になる可能性がある
   - 時間増分の選択には注意が必要

# 6-20 応答スペクトル法

1. 目的：
   地震に対する構造物の応答を考える場合、最も精度のよいとされる解析法

2. 特徴：
   - 構造物の基準面に強制変位として与え、過渡応答解析を行うことができる
   - 複雑な地震波を扱う場合、地震は一般に数秒から数分オーダーの現象のため、条件付き安定の陽解法でも十分すぎるほど小さな時間増分が必要

3. 解析手順：
   - フーリエ級数を用いて地震波を周期（振動数）ごとに分解
   - 各周期に対する最大応答値を求める
   - それらを包絡線で結んだものを応答スペクトルとする

4. 応答スペクトルの種類：
   - 変位応答スペクトル
   - 速度応答スペクトル
   - 加速度応答スペクトル

5. 計算式：
   $S_D = \frac{1}{\omega}\int_0^t \ddot{y}(τ)e^{-ζω(t-τ)} \sin ω(t-τ)dτ$
   $S_V = \int_0^t \ddot{y}(τ)e^{-ζω(t-τ)} \cos ω(t-τ)dτ$
   $S_A = ω\int_0^t \ddot{y}(τ)e^{-ζω(t-τ)} \sin ω(t-τ)dτ$

   ここで、ζは減衰比、ωは固有角振動数、$\ddot{y}$は地震波の加速度

6. 関係式：
   $S_D = \frac{1}{ω}S_V = \frac{1}{ω^2}S_A$

7. 最大値：
   加速度応答スペクトルは慣性力の最大値$f_{max}$を与えるための指標となる：
   $f_{max} = mS_1$

8. エネルギー関係：
   速度応答スペクトルは構造物のひずみエネルギーを表し、その最大値は：
   $\frac{1}{2}kv_{max}^2 = \frac{1}{2}(ωS_v)^2 = \frac{1}{2}ω^2S_D^2$

# 6-21 共振曲線

1. 運動方程式：
   $m\ddot{x}+c\dot{x}+kx=F\cos(ωt)$

   ここで、x, $\dot{x}$, $\ddot{x}$：質点の変位、速度、加速度
   m：質点の質量、c：減衰係数、k：ばね定数、F：外力の振幅

2. 無減衰系の固有振動数：
   $Ω = \sqrt{\frac{k}{m}}$

3. 減衰比：
   $ζ = \frac{c}{2\sqrt{km}} = \frac{c}{2mΩ}$

4. 定常振動解：
   $x = A\cos(ωt-α)$

5. 振幅比（応答倍率）：
   $M = \frac{A}{D} = \frac{1}{\sqrt{[1-(\frac{ω}{Ω})^2]^2 + [2ζ\frac{ω}{Ω}]^2}}$

6. 特性：
   - ζ < 1の場合、共振点付近で応答が急激に大きくなる
   - ζ = 0（無減衰系）の場合、ω = Ωで振幅は無限大になる
   - 減衰比が大きくなるにつれて振幅が小さくなる

7. 共振：
   ω/Ω = 1で無限大になる曲線（ζ = 0の場合）
